# 📍 Pathfinding Visualizer (Dijkstra's Algorithm)

**[🔗 View Live Demo](https://sneha811.github.io/pathfinding-visualizer/)** | **[📂 Source Code](https://github.com/sneha811/pathfinding-visualizer)**

---
A high-performance, web-based tool to visualize how **Dijkstra's Algorithm** calculates the shortest path between two points. 
Built with a focus on smooth UI/UX and efficient data structures.

---

## 🚀 Features
* **Interactive Grid:** Drag the 🟢 Start and 🔴 End nodes to any position.
* **Real-time Wall Drawing:** Click and drag to create obstacles.
* **Algorithm Visualization:** Watch the search expand and the path traced in real-time.
* **Optimized Performance:** Uses a custom **Min-Heap (Priority Queue)** for $O(E \log V)$ efficiency.

# 📍 Pathfinding Visualizer (Dijkstra's Algorithm)

A high-performance, web-based tool to visualize how **Dijkstra's Algorithm** calculates the shortest path between two points. Built with a focus on smooth UI/UX and efficient data structures.

---

## 🚀 Features
* **Interactive Grid:** Drag the 🟢 Start and 🔴 End nodes to any position.
* **Real-time Wall Drawing:** Click and drag to create obstacles.
* **Algorithm Visualization:** Watch the search expand and the path traced in real-time.
* **Optimized Performance:** Uses a custom **Min-Heap (Priority Queue)** for $O(E \log V)$ efficiency.

## 🧠 What I Learned
During this project, I deepened my understanding of:
1.  **Graph Theory:** Implementing Dijkstra's algorithm on a 2D grid.
2.  **State Management:** Handling mouse events and node dragging without UI lag.
3.  **Data Structures:** Why a Min-Heap is essential for optimizing priority queues in pathfinding.
